数据看板功能已实现

已完成的功能：

  • ✅ Dashboard Store (Pinia) - 统计数据状态管理
  • ✅ StatCard 组件 - 数据卡片展示
  • ✅ TrendChart 组件 - ECharts 趋势图表
  • ✅ PostsDashboard 组件 - 推文看板
  • ✅ AdsDashboard 组件 - 广告看板
  • ✅ Dashboard 页面 - 标签页切换功能

功能说明：

推文看板包含：

  • 推文总数、已发布、草稿、总浏览量统计卡片
  • 最近 7 天推文发布趋势折线图
  • 最受欢迎的推文 Top 10 列表（按浏览量排序）

广告看板包含：

  • 广告总数、活跃广告、总点击次数、总展示次数统计卡片
  • 平均点击率 (CTR) 显示
  • 最近 7 天广告点击率趋势柱状图
  • 表现最好的广告 Top 10 列表（按点击次数排序）

技术实现：

  • 使用 Pinia 进行状态管理
  • 使用 ECharts 6.0 进行数据可视化
  • 使用 Element Plus 组件库
  • 响应式设计，支持不同屏幕尺寸
  • 后端 API 提供完整的统计数据、趋势数据和排行榜数据
